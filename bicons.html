Getting Started

Most browsers support some basic symbols, with no dependencies. bIcons allows you to render native icons and symbols by applying classes! Keep in mind that various browsers may render symbols differently. Simply use the following markup, where "icon-name" is any of the below class names.

<i class="bi bi-icon-name"></>

Icon Sizes

Apply these classes to change the icon size. The .bi-fi class is fluid and resizes in proportion to the screen size.


  • .bi-2x

  • .bi-3x

  • .bi-4x

  • .bi-5x

  • .bi-6x

  • .bi-fi

The filetype icons library allows you to automatically append icons to lists of linked files. Over 400 filetypes are supported.
Simply load the filetypes CSS script from: /core/css/filetypes/filetypes.css
